WebAudio

WebAudio

Create a new WebAudio instance, to manage playback of a streaming music track.

new WebAudio(options: Object)
Parameters
options (Object)
Name Description
options.context AudioContext An instance of window.AudioContext.
options.throttleDecode [Number] (default 500) Number of milliseconds to throttle calls to the decode function, set this number too low and the browser will be tasked to decode many chunks, set it too high and there will be a delay after the chunk has been downloaded.
options.minLoadDuration [Number] (default 1) Minimum number of seconds required for a chunk of the track to be loaded, before it is played. This is useful to prevent a pause half a second after playback has started.
Properties
context (AudioContext)
audioBuffer (AudioBuffer)
audioSource (AudioBufferSourceNode)
loading (Boolean) : Is a file currently being downloaded?
paused (Boolean) : Is the audio track currently paused (i.e not playing)?
buffering (Boolean) : Have we reached the end of the buffer?
onProgress (Function) : Triggered when the audio buffer updates.
onLoad (Function) : Triggered when we finish decoding the entire file.
onPlay (Function) : Triggered after starting playback.
onPause (Function) : Triggered after pausing playback.
onStop (Function) : Triggered after stopping playback.
